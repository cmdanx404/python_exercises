Burger Order
Write a program that calculates final bill Burger Order Price based on a user's choice.

Price List.

Mini Burger (M) : $5

Normal Burger (N): $8

Large Burger (L) : $10

Add Mushroom : For Mini and Normal = $1, For Large = $2

Extra Cheese : $1



Example Input

size = "N"
add_mushroom = "Y"
extra_cheese = "N"
Example Output

Your final bill is: $9.
