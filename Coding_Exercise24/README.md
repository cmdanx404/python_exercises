Factorial using Loop

Instruction
Implement a function that takes an integer as a parameter and returns factorial of his number.

Factorial Equation(!)
5! = 5 x 4 x 3 x 2 x 1 = 120
4! = 4 x 3 x 2 x 1 = 24

conditions:
if input is 0 then the return value will be: The factorial of 0 is 1
if input is a negative integer return value will be: factorial does not exist for negative numbers
